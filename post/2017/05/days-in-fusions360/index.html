我在臺灣金融科技公司的那些日子

  • 2017-05-27
  • Liu, An-Chi 劉安齊
這篇文章紀錄我在大二時於臺灣金融科技公司實習工讀的心得，近期我選擇要離開公司，總覺得有點不捨，於是留下此文章作為昔日回顧此時之寫照。

任職公司： 臺灣金融科技股份有限公司 Fusions360
CEO： 王可言
職稱：工程實習生
時間：2016/11 ~ 2017/5

回顧一開始，是我想打工賺點錢，大一暑假幫教授寫軟體賺了2萬，可是買新電腦動輒3萬以上，所以就興起找工作的想法，因為覺得寫程式蠻好玩的，而且在企業兼職聽起來本身就是一件很屌的事，且做喜歡的是賺錢是多棒的一件事。

當時在一些平台上投履歷，大部分都無聲，後來剛好看到公司有在徵實習生，而且又是學校最近在接觸很夯的金融科技，更讓我想進公司了。投履歷後不久馬上有收到副總來信，找我去面試，面試過程就是我把我自己做過的幾個專案秀給公司看，沒有問我太多技術面的東西，只問我一些基礎技術會不會用，不會的話可以進公司在學，總之很順利，讓我以為找工作是一件很簡單的事。直到近期想投履歷，投了快二十家還沒下文才知道原來工作那麼難找。能順利進公司蠻幸運的，我想是因為當時第一次招收工程師的實習生，所以標準也沒有設到很高，看到有人能力還OK，就讓他進去了。不過我進去之後，再也沒有新的工程師進公司了。

公司編制大約是，CEO、VP、CTO、前端x2、協理x4、後端x6、業務x2、市場分析x2、行政x1、顧問若干，實習生若干。後來的實習生都是透過我在台大版PO文徵來的，都是進行市場研究，有一個是研究法律。公司有不少是從資策會過來的，因為董事長、副總當時就在資策會，所以帶了一些工程師出來創業。當初是董事長認為臺灣金融科技的腳步落世界了，而台灣如果新創從事金融科技可能會面臨市場不夠大以及技術的門檻限制，想要在台灣發展起金融科技似乎最可行的方式是建立一個金融科技的生態圈，所以CEO的理想就是建立一家公司，提供的服務可以串起整個金融科技的生態鏈。一個生態圈一定會比單打獨鬥，更有競爭力。

我一直覺得這邊的人都很厲害，像是CEO美國  IBM 20餘年，回國之後擔任資策會副執行長，副總則是一直是CEO的副手。CTO 在台灣IBM時是Band D 台灣區最高 CTO。有工程師有許多比賽得獎，還有申請專利。也有協理之前是 hTC 專案經理。也有個蒙古來的資料工程師。每個人都有自己的故事，中午大家常常會一起吃飯，跟大家聊天聽大家的故事都覺得很有趣，也覺得大家很厲害。每個人也都很好相處，大家也都會不吝嗇分享各種經驗，跟大家相處真的很快樂，這也是為甚麼我會覺得依依不捨的原因吧！

當初我是以前端工程師的身分進公司，被告知要用 Angular 2 來開發，所以就特別去研究了一翻，為此還寫了一系列的文章，寫完之後差不多就可以一般使用 Angular 2了。那時候公司原本要有做區塊鏈的錢包，是個小型專案所以就指派給我，照公司規範是用 Angular 2 開發，做完的確有對 Angular 更加熟習，但這個專案最後只成為一次金融發表會 Demo 用的專案而已，覺得很可惜。後來區塊鏈錢包開發完之後，想做點後端的東西，看到區塊鏈後端的部分缺人，就主動投入，原本區塊鏈後端是用 Node.js 寫的，但配合公司規範要轉成 Java，而我那時對 Java 一竅不通，雖然說程式語言大致上都很像，但又因為公司內部已經開發很多套件，但這些套件都沒有一個完整的說明文件，使用起來非常吃力，所以投入Java區塊鏈時非常吃力且一直卡住，過程中還研究了一下 Etherium 的內部結構。不過後來區塊鏈後端就收掉了，只繼續研究技術而不開發。接著又做了一些前端，然後投入後端做爬蟲，爬各種新聞，這次過程中就有真的 Java 的基礎好好建立起來，雖然也沒有完整的練起來，但至少可以照著自己意思操作。接著又做了幾個零碎的工作，最後離職前跟 CEO 談好要把建立專利資訊的程式寫好。

在公司的這段日子，我深深體會到一家新創公司如何經營，以前都是聽各種演講、看書、上課，那種體會完全不及我真正加入一家新創來的深刻。雖然說公司是新創，但因為公司內部的人在業界都很久了，很多制度面其實也沒真的很創新，有些地方還是不容易從舊有思維中打破，像IBM那樣的大企業出來創業，資源、環境勢必遠遠不及原來，如何調適心態、思維都是挑戰。在這裡也體會到創業真的很難，制定商業策略、設定公司制度、尋找市場還有一堆產品開發上的問題，我過去被各種成功的案例洗腦太多，真的以為有想法肯打拼就會殺出一條血路，但事實上真的難太多了，畢竟99% 新創經營不下去就是真實的數字。

而我待了半年後選擇離開，我真的很喜歡公司的環境、公司的人，但是我也看到公司的一些問題點，加上覺得學校課業有點吃不消，本來想做到暑假在走，但後來想提早休息。而且我想多多體驗各種產業，覺得不該在同一個地方待太久，畢竟我也還沒真到找到人生方向，我想多多體驗對我會是件好事。最後，我感謝臺灣金融科技公司以及公司的同仁對我的照顧，在這裡我成長很多！
